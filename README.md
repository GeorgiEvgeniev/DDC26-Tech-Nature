# DDC26-Tech-Nature

## 🚀 Running the Project

This project runs using a live server in Visual Studio Code.

### 1. Install Live Server
Install the **Live Server** extension in VS Code.

### 2. Start the Project
Right-click on `index.html` and select **"Open with Live Server"**.

### ⚠️ Notes
- A separate control window may open when the project starts.  
- If it does not appear, allow third-party pop-ups in your browser.

## 🎮 Controls

Make sure the control window is active.

- Use your **mouse** or **arrow keys**  
- Interact with the simulation of technology and nature
